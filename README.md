# Hotel Revenue Analytics – Power BI Dashboard

This project presents an **interactive Power BI dashboard** built to analyze hotel revenue performance across multiple properties. The report focuses on key hospitality KPIs and enables users to explore revenue trends through dynamic visuals and filters.

## Dashboard Overview

The report contains a single analytics page displaying important performance metrics and visual insights.

### Key KPIs

* Total Revenue
* ADR (Average Daily Rate)
* RevPAR (Revenue Per Available Room)
* Realization %
* Occupancy %
* DSRN (Daily Sellable Room Nights)

Each KPI includes **tooltip insights showing category-wise averages** for deeper analysis.

### Interactive Filters

Users can filter the dashboard using:

* City
* Room Category
* Month
* Week

### Visualizations

* **Donut Chart:** Revenue distribution by room category
* **Pie Chart:** ADR and Realization % by booking platform
* **Table Visual:** Property-level metrics including Property ID, Property Name, City, Revenue, ADR, BRN, URN, and SRN

## Purpose

The dashboard helps stakeholders quickly understand hotel performance, compare properties, and identify revenue patterns through interactive data exploration.

## Tools Used

* Power BI
* DAX Measures
* Interactive Filters & Tooltips
